# App Store Screenshots

This directory contains screenshots for App Store and Play Store submissions.

## Required Sizes

### iOS (App Store Connect)

| Device | Size | Directory |
|--------|------|-----------|
| iPhone 6.7" | 1290 x 2796 | `ios/iphone-6.7/` |
| iPhone 6.5" | 1284 x 2778 | `ios/iphone-6.5/` |
| iPhone 5.5" | 1242 x 2208 | `ios/iphone-5.5/` |
| iPad 12.9" | 2048 x 2732 | `ios/ipad-12.9/` |

### Android (Google Play Console)

| Device | Minimum Size | Directory |
|--------|--------------|-----------|
| Phone | 1080 x 1920 | `android/phone/` |
| 7" Tablet | 1200 x 1920 | `android/tablet-7/` |
| 10" Tablet | 1800 x 2560 | `android/tablet-10/` |

## Screenshot Requirements

### iOS
- 2-10 screenshots per device size
- PNG or JPEG format
- No transparency
- Must not include status bar or home indicator

### Android
- 2-8 screenshots per device type
- JPEG or 24-bit PNG
- Minimum 320px, Maximum 3840px per side
- 16:9 or 9:16 aspect ratio recommended

## Recommended Screenshots

1. **Home/Dashboard** - Main app view showing daily entry summary
2. **Voice Entry** - Voice recording interface
3. **Insights** - AI-generated insights and patterns
4. **Health Integration** - Health data correlations
5. **Conversation** - Chat interface with AI
6. **Settings** - Customization options

## Naming Convention

Name files with a number prefix for ordering:
- `01_home.png`
- `02_voice_entry.png`
- `03_insights.png`
- etc.
